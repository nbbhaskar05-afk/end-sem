⚡ Laplace Transforms
Definition
L{f(t)} = ∫₀^∞ e^(-st) f(t) dt
First Shifting Theorem
L{e^(at)f(t)} = F(s-a)
Second Shifting (t-shifting)
L{f(t-a)·u(t-a)} = e^(-as)·F(s)
Laplace of Derivatives
L{f'(t)} = sF(s) - f(0)
L{f''(t)} = s²F(s) - sf(0) - f'(0)
Convolution Theorem
L{f*g} = F(s)·G(s)
(f*g)(t) = ∫₀^t f(τ)g(t-τ)dτ
Unit Step Function
L{u(t-a)} = e^(-as)/s
u(t-a) = 0 if t<a, 1 if t≥a
Dirac Delta
L{δ(t-a)} = e^(-as)
Key Standard Pairs
L{1}=1/s, L{t^n}=n!/s^(n+1)
L{e^(at)}=1/(s-a)
L{sin(at)}=a/(s²+a²)
L{cos(at)}=s/(s²+a²)
Differentiation of Transform
L{t·f(t)} = -d/ds[F(s)]
L{t^n·f(t)} = (-1)^n · F^(n)(s)
Integration of Transform
L{f(t)/t} = ∫_s^∞ F(u)du
〜 Fourier Series & Transform
Fourier Series (period 2L)
f(x) = a₀/2 + Σ[aₙcos(nπx/L) + bₙsin(nπx/L)]
a₀=1/L∫f(x)dx, aₙ=1/L∫f(x)cos(nπx/L)dx
bₙ=1/L∫f(x)sin(nπx/L)dx
Even Function (Cosine Series)
bₙ = 0
aₙ = 2/L ∫₀^L f(x)cos(nπx/L)dx
Odd Function (Sine Series)
aₙ = 0
bₙ = 2/L ∫₀^L f(x)sin(nπx/L)dx
Fourier Transform
F(ω) = ∫_{-∞}^{∞} f(t)e^(-iωt) dt
f(t) = 1/2π ∫F(ω)e^(iωt) dω
Fourier Cosine Transform
Fc(ω) = √(2/π) ∫₀^∞ f(x)cos(ωx)dx
Fourier Sine Transform
Fs(ω) = √(2/π) ∫₀^∞ f(x)sin(ωx)dx
Parseval's Identity
∫|f(x)|² dx = 1/2π ∫|F(ω)|² dω
🔢 Numerical Root Finding
Bisection Method
x_new = (a + b) / 2
If f(a)·f(x_new) < 0 → root in [a, x_new] else [x_new, b]
Newton-Raphson
x_{n+1} = x_n - f(x_n)/f'(x_n)
Order of convergence: 2 (quadratic)
Secant Method
x_{n+1} = x_n - f(x_n)·(x_n - x_{n-1}) / (f(x_n) - f(x_{n-1}))
Order of convergence: 1.618 (superlinear)
Regula-Falsi
x = [a·f(b) - b·f(a)] / [f(b) - f(a)]
Fixed Point Iteration
x_{n+1} = g(x_n)
Converges if |g'(x)| < 1
📈 Interpolation
Newton's Forward
f(x) = f(x₀) + p·Δf₀ + p(p-1)/2!·Δ²f₀ + ...
p = (x - x₀)/h
Newton's Backward
f(x) = f(xₙ) + s·∇f_n + s(s+1)/2!·∇²f_n + ...
s = (x - xₙ)/h
Lagrange Interpolation
f(x) = Σ Lₖ(x)·f(xₖ)
Lₖ(x) = Π(x-xⱼ)/(xₖ-xⱼ), j≠k
Divided Difference
f[x₀,x₁] = (f(x₁)-f(x₀))/(x₁-x₀)
f[x₀,x₁,x₂] = (f[x₁,x₂]-f[x₀,x₁])/(x₂-x₀)
Forward Difference Operators
Δf(x) = f(x+h) - f(x)
∇f(x) = f(x) - f(x-h)
δf(x) = f(x+h/2) - f(x-h/2)
∫ Numerical Integration
Trapezoidal Rule
∫ ≈ h/2·[f₀ + 2(f₁+...+f_{n-1}) + fₙ]
Error = -h²/12 · (b-a) · f''(ξ)
Simpson's 1/3 Rule
∫ ≈ h/3·[f₀ + 4f₁ + 2f₂ + 4f₃ + ... + fₙ]
n must be even. Error ∝ h⁴
Simpson's 3/8 Rule
∫ ≈ 3h/8·[f₀ + 3f₁ + 3f₂ + 2f₃ + ... + fₙ]
n must be multiple of 3
Gauss-Legendre 2-point
∫₋₁¹ f(x)dx ≈ f(-1/√3) + f(1/√3)
Points: ±0.5774, Weights: 1, 1
Gauss-Legendre 3-point
∫₋₁¹ f(x)dx ≈ 5/9·f(-√(3/5)) + 8/9·f(0) + 5/9·f(√(3/5))
Points: 0, ±0.7746 | Weights: 8/9, 5/9, 5/9
🔄 Numerical ODE Methods
Euler's Method
y_{n+1} = yₙ + h·f(xₙ, yₙ)
Modified Euler (Heun's)
y* = yₙ + h·f(xₙ,yₙ)
y_{n+1} = yₙ + h/2·[f(xₙ,yₙ) + f(x_{n+1},y*)]
RK4 — Full Formula
k₁ = h·f(xₙ, yₙ)
k₂ = h·f(xₙ+h/2, yₙ+k₁/2)
k₃ = h·f(xₙ+h/2, yₙ+k₂/2)
k₄ = h·f(xₙ+h, yₙ+k₃)
y_{n+1} = yₙ + 1/6·(k₁+2k₂+2k₃+k₄)
Gauss-Jacobi
x_i^(k+1) = 1/a_ii · [b_i - Σ_{j≠i} a_ij · x_j^(k)]
All values from previous iteration used
Gauss-Seidel
x_i^(k+1) = 1/a_ii · [b_i - Σ_{j<i} a_ij·x_j^(k+1) - Σ_{j>i} a_ij·x_j^(k)]
Uses updated values immediately (faster)
Numerical Differentiation
f'(x) ≈ Δf/h (forward)
f''(x) ≈ Δ²f/h²
= [f(x+h)-2f(x)+f(x-h)]/h²
